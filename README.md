# 🌟 What is LangGraph?

LangGraph is a framework used to build:

👉 Stateful AI systems
👉 Multi-step workflows
👉 Agent-based pipelines

Unlike normal LLM apps, it allows:

✔ Memory
✔ Decision-making
✔ Looping (thinking process)

---

# ❓ Why Use LangGraph?

Traditional AI systems:

* One input → One output
* No memory
* No control over flow

LangGraph provides:

✔ Control over execution flow
✔ Persistent state
✔ Multi-agent collaboration
✔ Dynamic decision making

# ⚙️ Core Concepts Covered

## 🧾 State

* Shared memory across nodes
* Maintains context

## 🔧 Nodes

* Functions or LLM calls
* Perform specific tasks

## 🔀 Edges

* Define workflow direction
* Enable conditional routing

## 🔁 Loops

* Allow iterative reasoning
* Core of Agentic AI

## 🚪 Entry & Exit

* Start and end of execution
